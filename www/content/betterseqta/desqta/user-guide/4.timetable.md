## Timetable Overview

The Timetable page displays your complete class schedule in an interactive calendar format. It shows all your lessons, classes, and activities with subject-specific colors for easy identification.

## Accessing Your Timetable

Navigate to Timetable from:
- **Sidebar Menu**: Click "Timetable" in the navigation sidebar
- **Dashboard**: Click "Today's Schedule" widget
- **Direct URL**: `/timetable`

## Timetable Views

### Day View

Shows all classes for a single day in a detailed time grid.

**Features:**
- Hour-by-hour breakdown
- Current time indicator
- Color-coded by subject
- Click classes for details

**Navigation:**
- Use arrow buttons to move between days
- Click date picker to jump to specific date
- "Today" button returns to current day

### Week View

Displays your full weekly schedule in a grid format.

**Features:**
- All days of the week visible
- Time slots on left side
- Color-coded subjects
- Easy week navigation

**Best For:**
- Weekly planning
- Seeing weekly patterns
- Understanding routine

### Month Grid View

A traditional calendar grid showing your schedule.

**Features:**
- Full month overview
- Classes shown as colored blocks
- Click dates to see details
- Month navigation

**Best For:**
- Long-term planning
- Seeing monthly overview
- Planning ahead

### Month Agenda View

A list-style view of your month's classes.

**Features:**
- Chronological list
- All classes for the month
- Grouped by date
- Easy scanning

**Best For:**
- Finding specific classes
- Seeing all upcoming classes
- Quick reference

## Understanding Timetable Colors

Each subject has a unique color assigned in SEQTA. These colors are used throughout DesQTA for consistency:

- **Subject Colors**: Match your SEQTA timetable colors
- **Consistent Across App**: Same colors in assessments, courses, etc.
- **Customizable**: Can be changed in SEQTA preferences

## Class Information

Click any class to see detailed information:

### Basic Details
- **Subject**: Course name
- **Time**: Start and end time
- **Date**: Class date
- **Location**: Room or location
- **Teacher**: Instructor name

### Additional Information
- **Class Code**: Subject code
- **Description**: Class description (if available)
- **Resources**: Attached materials
- **Notes**: Your personal notes

## Timetable Features

### Current Time Indicator

A visual line shows the current time in day and week views, making it easy to see:
- What class you're in now
- What's coming up next
- How much time until next class

### Subject Color Coding

All classes are color-coded by subject:
- Consistent with SEQTA colors
- Easy visual identification
- Helps spot patterns

### Navigation Controls

Easy navigation between time periods:
- **Previous/Next**: Move backward or forward
- **Today**: Jump to current day
- **Date Picker**: Select specific date
- **View Switcher**: Change view type

### Responsive Design

The timetable adapts to your screen:
- **Desktop**: Full calendar views
- **Tablet**: Optimized layout
- **Mobile**: Stacked, touch-friendly interface

## Exporting Your Timetable

### Export as Calendar File

1. Click "Export" button
2. Choose "Export as ICS"
3. File downloads
4. Import into your calendar app (Google Calendar, Outlook, etc.)

### Calendar Integration

Your timetable can be synced with:
- Google Calendar
- Apple Calendar
- Outlook
- Any calendar app supporting ICS format

## Troubleshooting

### Missing Classes
- Refresh the page
- Check if date range is correct
- Verify SEQTA connection
- Contact school IT if persistent

### Wrong Times
- Times come from SEQTA
- Check SEQTA directly to verify
- Report discrepancies to teacher

### Colors Not Showing
- Colors sync from SEQTA preferences
- Check SEQTA color settings
- Refresh to reload colors

### Export Not Working
- Check browser download permissions
- Try different browser
- Verify file isn't blocked
