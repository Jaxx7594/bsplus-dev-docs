## Overview

The Today's Schedule widget displays all your classes and lessons scheduled for today in a clean, easy-to-read format. It's color-coded by subject and shows important details like time, location, and teacher.

## Widget Features

### Display Information

Each class shows:
- **Subject Name** - Full course name
- **Subject Code** - Short code (e.g., "MATH101")
- **Time** - Start and end times
- **Location** - Room or location
- **Teacher** - Instructor name
- **Color** - Subject-specific color from your timetable

### Current Class Highlighting

The widget automatically highlights:
- **Current Class** - The class you're in right now (if any)
- **Next Class** - Your upcoming class
- **Visual Indicators** - Clear visual distinction

### Date Navigation

- **Previous Day** - Click left arrow to see yesterday
- **Next Day** - Click right arrow to see tomorrow
- **Today Button** - Jump back to today's schedule
- **Date Display** - Shows selected date clearly

## Using the Widget

### Viewing Today's Schedule

1. **Dashboard:** Widget appears automatically on dashboard
2. **Current Day:** Shows today's classes by default
3. **Scroll:** Scroll through classes if you have many

### Navigating Days

**To see a different day:**
1. Click **left arrow** (◀) for previous day
2. Click **right arrow** (▶) for next day
3. Click **"Today"** button to return to current day

### Viewing Class Details

**Click any class** to:
- See full class information
- View class description
- Access class resources
- Navigate to full timetable page

## Widget Behavior

### Auto-Update

- **Real-Time:** Updates every minute to show current class
- **Date Changes:** Automatically updates at midnight
- **Manual Refresh:** Refresh page to get latest data

### Color Coding

- **Subject Colors:** Matches your SEQTA timetable colors
- **Consistent:** Same colors used throughout DesQTA
- **Customizable:** Change colors in SEQTA preferences

### Empty States

**If no classes today:**
- Widget shows "No classes scheduled"
- Still allows navigation to other days
- Click to view full timetable

## Integration with Timetable

### Quick Access

- **Click Widget:** Opens full timetable page
- **View Full Schedule:** See week/month views
- **Export:** Export timetable from full page

### Data Source

- **SEQTA Integration:** Pulls from your SEQTA timetable
- **Real-Time Sync:** Updates from SEQTA automatically
- **Cached:** Uses caching for faster loading

### Daily Planning

2. **Time Management:** Plan breaks between classes
3. **Preparation:** Know what's coming up next

### Navigation

- **Quick Navigation:** Use arrows to check tomorrow/yesterday

## Troubleshooting

### Classes Not Showing

**Check:**
- Correct date selected
- Classes exist in SEQTA for that day
- Internet connection active

**Solutions:**
- Navigate to different day
- Refresh the page
- Check SEQTA directly

### Wrong Times

**Verify:**
- Times come from SEQTA
- Check SEQTA timetable directly
- Report discrepancies to teacher

### Colors Not Matching

**Fix:**
- Colors sync from SEQTA preferences
- Update colors in SEQTA settings
- Refresh DesQTA to reload colors

### Widget Not Updating

**Try:**
- Refresh the page
- Check internet connection
- Verify SEQTA session is active
