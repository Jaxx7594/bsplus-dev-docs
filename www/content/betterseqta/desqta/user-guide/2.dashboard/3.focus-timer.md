## Overview

The Focus Timer widget is a Pomodoro-style study timer that helps you maintain focus during study sessions. It alternates between focused work periods and breaks to maximize productivity.

## How Pomodoro Technique Works

The Pomodoro Technique is a time management method:

1. **Work Session:** Focus for 25 minutes (default)
2. **Short Break:** Rest for 5 minutes
3. **Repeat:** Continue cycle
4. **Long Break:** After 4 cycles, take 15-minute break

## Widget Features

### Timer Display

- **Large Display:** Easy-to-read countdown timer
- **Visual Indicator:** Color changes for work vs break
- **Time Format:** MM:SS format (e.g., 25:00)

### Timer Controls

- **Start:** Begin timer countdown
- **Pause:** Pause timer (resume later)
- **Reset:** Reset to selected duration

### Duration Options

**Preset Durations:**
- **25 minutes** - Standard Pomodoro (default)
- **15 minutes** - Short session
- **45 minutes** - Extended session
- **Custom** - Set your own duration

**Break Durations:**
- **5 minutes** - Short break (default)
- **15 minutes** - Long break (after 4 cycles)

### Visual States

**Work Mode:**
- Accent color border
- Focused appearance
- Work indicator

**Break Mode:**
- Green color border
- Relaxed appearance
- Break indicator

## Using the Focus Timer

### Starting a Session

1. **Select Duration:** Choose 15, 25, or 45 minutes (or custom)
2. **Click Start:** Timer begins countdown
3. **Focus:** Work on your task
4. **Timer Alerts:** Notification when time is up

### During a Session

- **Timer Counts Down:** Watch time decrease
- **Stay Focused:** Work on your task
- **Pause if Needed:** Click Pause for interruptions
- **Resume:** Click Start to continue

### When Timer Ends

**Automatic Actions:**
- **Notification Sound:** Audio alert plays
- **Visual Alert:** Timer flashes/animates
- **Break Starts:** Automatically switches to break mode
- **Break Timer:** 5-minute break begins

### Break Period

**During Break:**
- Timer shows break countdown
- Green indicator shows break mode
- Rest and recharge
- Prepare for next session

**After Break:**
- Timer automatically starts next work session
- Or manually start when ready

## Customization

### Setting Custom Duration

1. **Click Custom:** Select custom duration option
2. **Enter Minutes:** Type desired minutes
3. **Enter Seconds:** Type desired seconds (optional)
4. **Start:** Begin custom timer

### Timer Settings

**Available Options:**
- **Work Duration:** 15, 25, 45 minutes or custom
- **Break Duration:** 5 minutes (short) or 15 minutes (long)
- **Sound Notifications:** Enable/disable audio alerts

## Troubleshooting

### Timer Not Starting

**Check:**
- Duration is selected
- Click Start button
- No errors in console

**Solutions:**
- Select a duration
- Click Start again
- Refresh widget if needed

### Sound Not Playing

**Check:**
- Browser permissions for audio
- System volume is up
- Notification sounds enabled

**Solutions:**
- Allow audio in browser settings
- Check system volume
- Enable notifications

### Timer Not Switching to Break

**Check:**
- Timer reached zero
- Auto-switch is enabled
- No errors occurred

**Solutions:**
- Wait for timer to finish
- Manually switch to break
- Refresh if stuck

### Custom Duration Not Working

**Verify:**
- Valid numbers entered
- Minutes between 1-60
- Seconds between 0-59

**Solutions:**
- Enter valid duration
- Use preset durations if issues
- Check format (MM:SS)
