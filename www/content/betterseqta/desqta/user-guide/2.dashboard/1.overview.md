## What is the Dashboard?

The Dashboard is your personalized homepage in DesQTA. It provides a comprehensive overview of your academic life at a glance, with customizable widgets that display the information most important to you.

## Dashboard Layout

The dashboard uses a responsive grid layout that adapts to your screen size:

### Layout Structure

```
┌─────────────────────────────────────────────────┐
│  Recent News  │  Messages Preview              │
├─────────────────────────────────────────────────┤
│  Today's Schedule (Full Width)                 │
├─────────────────────────────────────────────────┤
│  Quick Links (Full Width)                      │
├─────────────────────────────────────────────────┤
│  Notices (Full Width)                          │
├─────────────────────────────────────────────────┤
│  Upcoming Assessments (Full Width)             │
├─────────────────────────────────────────────────┤
│  Welcome Portal (Full Width)                   │
├─────────────────────────────────────────────────┤
│  Homework │ Todo List │ Focus Timer            │
└─────────────────────────────────────────────────┘
```

### Responsive Behavior

- **Desktop (Large Screens):** Widgets arranged in optimal grid
- **Tablet:** Some widgets stack vertically
- **Mobile:** All widgets stack vertically for easy scrolling

## Dashboard Widgets

Your dashboard includes these widgets (in order):

1. **Recent News** - Latest news articles
2. **Messages Preview** - Quick access to DireQt messages
3. **Today's Schedule** - Your classes for today
4. **Quick Links** - Customizable shortcuts
5. **Notices** - Important school announcements
6. **Upcoming Assessments** - Assessments due soon
7. **Welcome Portal** - Embedded portal content
8. **Homework** - Track homework assignments
9. **Todo List** - Personal task management
10. **Focus Timer** - Pomodoro-style study timer

## Accessing the Dashboard

Navigate to Dashboard from:
- **Sidebar Menu:** Click "Dashboard" (home icon) at the top
- **Direct URL:** `/` (root path)
- **Keyboard Shortcut:** `Alt+D` or `Cmd+D` (if configured)

## Dashboard Features

### Auto-Refresh

Dashboard widgets automatically refresh:
- **On Load:** Fresh data when you visit
- **Periodic Updates:** Every 10-30 minutes (varies by widget)
- **Manual Refresh:** Pull down on mobile or click refresh icon

### Caching

Dashboard uses intelligent caching:
- **Fast Loading:** Cached data loads instantly
- **Background Sync:** Updates happen in background
- **Offline Support:** Shows cached data when offline

### Customization

Personalize your dashboard:
- **Widget Visibility:** Show/hide widgets in Settings
- **Quick Links:** Add your own shortcuts
- **Layout:** Automatic responsive layout (custom ordering coming soon)

## Quick Actions

From the dashboard, you can:

- **View Full Details:** Click any widget to see more
- **Navigate:** Click links to go to full pages
- **Interact:** Use widgets directly (timer, todo list, etc.)
- **Refresh:** Manually refresh any widget

## Troubleshooting

### Widgets Not Loading

**Check:**
- Internet connection
- SEQTA session is active
- Widget isn't disabled in Settings

**Solutions:**
- Refresh the page
- Check Settings → Dashboard for disabled widgets
- Verify SEQTA connection

### Missing Data

**Possible causes:**
- Widget requires specific SEQTA permissions
- Data not available in SEQTA
- Cache needs refresh

**Solutions:**
- Refresh the widget
- Check SEQTA directly
- Clear cache in Settings

### Slow Performance

**Optimize:**
- Disable unused widgets
- Clear cache periodically
- Check internet connection
- Update DesQTA to latest version
