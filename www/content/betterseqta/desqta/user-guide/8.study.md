## Study Page Overview

The Study page is your personal productivity hub, combining task management, note-taking, and study resources in one place. It helps you stay organized and focused on your academic goals.

## Accessing Study Tools

Navigate to Study from:
- **Sidebar Menu**: Click "Study" in the navigation sidebar
- **Direct URL**: `/study`

## Study Page Tabs

The Study page has two main tabs:

### Tasks Tab

Your personal task management system with:
- Todo items
- Subtasks
- Due dates
- Priority levels
- Subject and assessment linking

### Notes Tab

Your personal note-taking space with:
- Rich text notes
- Subject organization
- Search functionality
- Quick access

## Task Management

### Creating Tasks

1. Click the **"+"** button or "Add Task" field
2. Enter task title
3. Press Enter to create, or click to add details

### Task Details

Each task can include:

#### Basic Information
- **Title**: Task name (required)
- **Description**: Additional details
- **Due Date**: When the task is due
- **Due Time**: Specific time (optional)
- **Priority**: Low, Medium, or High

#### Organization
- **Subject**: Link to a SEQTA subject
- **Assessment**: Link to a related assessment
- **Tags**: Custom tags for organization

#### Subtasks
- Break down tasks into smaller steps
- Check off subtasks as you complete them
- Visual progress indicator

### Editing Tasks

1. Click on a task to edit
2. Modify any field
3. Changes save automatically

### Completing Tasks

- **Checkbox**: Click the checkbox to mark complete
- **Visual Feedback**: Completed tasks fade out
- **Archive**: Completed tasks move to completed filter

### Deleting Tasks

1. Click the delete icon (trash) on a task
2. Confirm deletion
3. Task is permanently removed

## Task Filtering

### Filter Options

- **All**: Show all tasks
- **Today**: Tasks due today
- **Week**: Tasks due this week
- **Completed**: Finished tasks

### Sorting

Sort tasks by:
- **Due Date**: Chronological order
- **Priority**: High to low priority
- **Updated**: Recently modified first

### Search

Use the search bar to find tasks by:
- Title
- Description
- Subject
- Tags

## Subject and Assessment Linking

### Linking to Subjects

1. Start typing a subject code or name
2. Select from autocomplete suggestions
3. Task is linked and color-coded

### Linking to Assessments

1. Start typing an assessment title
2. Select from autocomplete suggestions
3. Task shows assessment due date
4. Quick link to assessment details

### Benefits of Linking

- **Color Coding**: Tasks inherit subject colors
- **Quick Access**: Jump to subject or assessment
- **Organization**: Filter by subject or assessment
- **Context**: See related information

## Notes Management

### Creating Notes

1. Switch to **Notes** tab
2. Click "New Note" or "+" button
3. Start typing your note

### Note Features

- **Rich Text**: Formatting options available
- **Subject Tags**: Link notes to subjects
- **Search**: Find notes quickly
- **Organization**: Group by subject or date

### Editing Notes

1. Click on a note to open
2. Edit content directly
3. Changes save automatically

### Deleting Notes

1. Open the note
2. Click delete button
3. Confirm deletion

## Troubleshooting

### Tasks Not Saving

**Check:**
- App has storage permissions
- Storage space available
- App is up to date

**Solutions:**
- Restart the app
- Check storage permissions
- Clear cache if needed

### Subjects Not Linking

**Possible causes:**
- Subject not loaded yet
- Typo in subject code
- Subject not available

**Solutions:**
- Wait for subjects to load
- Check spelling
- Use autocomplete suggestions

### Notes Not Loading

**Check:**
- Notes tab is selected
- Data is saved
- App permissions

**Solutions:**
- Refresh the page
- Check local storage
- Restart app

### Performance Issues

**Optimize:**
- Limit number of tasks
- Archive completed tasks
- Clear old notes
- Update app
