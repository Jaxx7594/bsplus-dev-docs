## Settings Overview

The Settings page is your control center for customizing DesQTA. Here you can adjust appearance, behavior, integrations, and more to make DesQTA work exactly how you want.

## Accessing Settings

Navigate to Settings from:
- **Sidebar Menu**: Click "Settings" at the bottom of the sidebar
- **User Menu**: Click your profile picture → Settings
- **Direct URL**: `/settings`

## Appearance Settings

### Theme Selection

Choose your preferred color scheme:

**Options:**
- **Light**: Light background, dark text
- **Dark**: Dark background, light text
- **System**: Follows your operating system preference

**How to Change:**
1. Go to Settings → Appearance
2. Select your preferred theme
3. Changes apply immediately

### Accent Color

Customize the accent color used throughout the app:

1. Go to Settings → Appearance
2. Click the color picker
3. Choose your preferred color
4. See preview in real-time

**Accent Color Usage:**
- Interactive elements (buttons, links)
- Focus indicators
- Highlights and selections
- Progress indicators

### Custom Themes

DesQTA includes multiple built-in themes:

- **Default**: Standard DesQTA theme
- **Aero**: Glass-like appearance
- **Bubblegum**: Bright, playful colors
- **Glass**: Transparent, modern look
- **Grape**: Purple-themed
- **Light**: Clean light theme
- **Midnight**: Deep dark theme
- **Mint**: Fresh green theme
- **Neon Cyber**: Futuristic neon style
- **Pink Dream**: Soft pink theme
- **Solarized**: Developer-friendly colors
- **Sunset**: Warm orange theme

**Applying a Theme:**
1. Go to Settings → Theme Store
2. Browse available themes
3. Click "Apply" on your chosen theme
4. Theme applies immediately

### Theme Builder

Create your own custom theme:

1. Go to Settings → Theme Store → Theme Builder
2. Customize colors, fonts, and styles
3. Preview changes in real-time
4. Save your custom theme
5. Share with others (optional)

## Dashboard Settings

### Widget Configuration

Control which widgets appear on your dashboard:

**Available Widgets:**
- Recent News
- Messages Preview
- Today's Schedule
- Quick Links
- Notices
- Upcoming Assessments
- Welcome Portal
- Homework
- Todo List
- Focus Timer

**Managing Widgets:**
1. Go to Settings → Dashboard
2. Toggle widgets on/off
3. Changes save automatically

### Quick Links

Add and manage shortcuts on your dashboard:

**Adding a Link:**
1. Go to Settings → Dashboard → Quick Links
2. Click "Add Shortcut"
3. Enter:
   - Name (e.g., "Outlook")
   - URL (e.g., "https://outlook.office.com")
   - Icon (emoji or icon name)
4. Click "Save"

**Managing Links:**
- Drag to reorder
- Click edit icon to modify
- Click delete icon to remove

## Notification Settings

### Assessment Reminders

Get notified about upcoming assessments:

**Options:**

- Set reminder timing (days before due)

**Settings:**
1. Go to Settings → Notifications
2. Toggle "Assessment Reminders"
3. Set reminder timing
4. Choose notification methods

### Message Notifications

Control DireQt message notifications:

- Enable/disable notifications
- Sound preferences
- Badge counts
- Notification style

## Weather Widget

### Enabling Weather

Display current weather in the header:

1. Go to Settings → Weather
2. Toggle "Enable Weather"
3. Enter your city and country
4. Weather appears in header

### Weather Options

- **Use Location**: Automatically detect location
- **Manual Entry**: Enter city and country
- **Update Frequency**: How often to refresh

## Sidebar Settings

### Auto-Collapse Sidebar

Automatically hide sidebar when navigating:

1. Go to Settings → Sidebar
2. Toggle "Auto-collapse on navigation"
3. Sidebar closes when clicking menu items

### Auto-Expand on Hover

Show sidebar when hovering near edge:

1. Go to Settings → Sidebar
2. Toggle "Auto-expand on hover"
3. Sidebar appears when mouse near left edge

## Advanced Settings

### Enhanced Animations

Enable smooth animations throughout the app:

1. Go to Settings → Advanced
2. Toggle "Enhanced Animations"
3. Enjoy smoother transitions

**Note:** Disable if experiencing performance issues.

### School Picture

Control profile picture display:

- **Show School Picture**: Display SEQTA profile photo
- **Hide School Picture**: Use default avatar instead

### Performance Settings

Optimize app performance:

- **Cache Management**: Clear cached data
- **Offline Mode**: Force offline mode for testing
- **Performance Testing**: Run performance diagnostics

## Cloud Sync Settings

### BetterSEQTA Cloud Account

Sync settings across devices:

**Setting Up:**
1. Go to Settings → Cloud Sync
2. Click "Sign In" or "Create Account"
3. Follow authentication steps
4. Settings sync automatically

**Features:**
- Sync settings across devices
- Backup your configuration
- Restore settings easily

### Cloud Sync Options

- **Auto-Sync**: Automatically sync changes
- **Manual Sync**: Sync on demand
- **Sync Frequency**: How often to sync

## RSS Feeds

### Adding RSS Feeds

1. Go to Settings → RSS Feeds

## Developer Settings

### Dev Mode

Advanced settings for developers:

- **Sensitive Info Hider**: Hide personal info in dev mode
- **Force Offline Mode**: Test offline functionality
- **Performance Testing**: Run diagnostics

**Warning:** Only enable if you know what you're doing.

## Exporting & Importing Settings

### Export Settings

Save your settings as a backup:

1. Go to Settings → Advanced
2. Click "Export Settings"
3. JSON file downloads
4. Keep as backup

### Import Settings

Restore settings from backup:

1. Go to Settings → Advanced
2. Click "Import Settings"
3. Select JSON file
4. Settings restore

## Resetting Settings

### Reset to Defaults

Restore all settings to defaults:

1. Go to Settings → Advanced
2. Click "Reset All Settings"
3. Confirm action
4. Settings reset

**Warning:** This cannot be undone!

## Troubleshooting Settings

### Settings Not Saving
- Check internet connection
- Verify app permissions
- Try clearing cache
- Restart app

### Theme Not Applying
- Refresh page
- Clear browser cache
- Check theme file permissions
- Try different theme

### Cloud Sync Issues
- Verify account login
- Check internet connection
- Try manual sync
- Contact support if persistent
