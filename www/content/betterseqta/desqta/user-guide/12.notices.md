## Notices Overview

The Notices page displays important school announcements, updates, and information. It's your central hub for staying informed about school events, deadlines, and important messages from teachers and administrators.

## Accessing Notices

Navigate to Notices from:
- **Sidebar Menu**: Click "Notices" in the navigation sidebar
- **Dashboard**: Click "Notices" widget
- **Direct URL**: `/notices`

## Notice Display

### Notice Cards

Each notice is displayed as a card showing:
- **Title**: Notice headline
- **Subtitle**: Additional context
- **Author**: Who posted the notice
- **Label**: Category or tag (color-coded)
- **Content**: Full notice text
- **Date**: When it was posted

### Notice Layout

- **Grid View**: Multiple notices visible
- **Card Design**: Easy to scan
- **Color Coding**: Labels help identify categories
- **Responsive**: Adapts to screen size

## Filtering Notices

### By Label

Filter notices by category:

1. Click the label dropdown
2. Select a label/category
3. Notices filter automatically
4. Select "All" to show everything

**Label Features:**
- **Color Coded**: Each label has a unique color
- **Quick Filter**: Easy category selection
- **Multiple Labels**: Notices can have multiple labels

### By Date

Filter notices by date:

1. Use the date selector
2. Choose a specific date
3. View notices from that day
4. Navigate between dates

**Date Features:**
- **Calendar Picker**: Easy date selection
- **Today's Notices**: Quick access to today
- **Historical**: View past notices
- **Upcoming**: See future notices (if available)

## Troubleshooting

### Notices Not Loading

**Check:**
- Internet connection
- SEQTA session is active
- School has notices enabled

**Solutions:**
- Refresh the page
- Check SEQTA directly
- Verify login status
- Contact IT support

### Missing Notices

**Possible causes:**
- Filter hiding notices
- Date filter too restrictive
- Notices not published yet

**Solutions:**
- Clear filters
- Check date range
- Verify with teachers
- Check SEQTA directly

### Labels Not Showing

**Check:**
- Labels are configured
- Filter is set correctly
- Notices have labels assigned

**Solutions:**
- Select "All" labels
- Check label configuration
- Verify notice has labels
- Contact administrator

### Search Not Working

**Possible causes:**
- Search term too specific
- Notices not indexed
- Browser issues

**Solutions:**
- Try broader terms
- Clear search and retry
- Refresh page
- Check browser compatibility
