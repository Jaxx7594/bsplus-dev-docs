## Courses Overview

The Courses page provides access to all your subject materials, including course content, lesson schedules, resources, and files. It's organized by subject folders, making it easy to find what you need for each class.

## Accessing Courses

Navigate to Courses from:
- **Sidebar Menu**: Click "Courses" in the navigation sidebar
- **Direct URL**: `/courses`

## Course Structure

Courses are organized hierarchically:

### Folders

Subjects are grouped into folders:
- **Active Folders**: Current subjects you're enrolled in
- **Other Folders**: Past subjects or archived courses

### Subjects

Each folder contains one or more subjects:
- **Subject Code**: Short identifier (e.g., "ENG", "MATH")
- **Subject Title**: Full name of the subject
- **Description**: Course description and details
- **Color Coding**: Each subject has a unique color for visual identification

## Viewing Course Content

### Selecting a Subject

1. Navigate to the Courses page
2. Expand a folder by clicking on it
3. Click on a subject to view its content

### Course Overview

When you select a subject, you'll see:

- **Course Title**: Name of the course
- **Course Image**: Subject or course image
- **Main Content**: Course description and information
- **Course Files**: Available resources and documents
- **Lesson Schedule**: Weekly lesson plan

### Course Content Sections

#### Overview Tab

The default view showing:
- Course description
- Main course content modules
- Links to external resources
- Embedded content

#### Schedule Tab

View lessons organized by:
- **Term**: Academic term
- **Week**: Week number within the term
- **Lessons**: Individual lesson entries with:
  - Period number
  - Date and time
  - Lesson content
  - Attached resources

### Navigating Lessons

1. Click the **Schedule** tab
2. Select a term from the dropdown
3. Browse weeks using navigation arrows
4. Click on a lesson to view details

## Course Modules

Course content is organized into modules:

### Title Modules

Section headers and titles within the course content.

### Text Block Modules

Rich text content including:
- Formatted text
- Links
- Embedded content
- Styled sections

### Resource Modules

Collections of downloadable files:
- Documents
- PDFs
- Images
- Other resources

### Link Modules

External links to:
- Websites
- Online resources
- Related content

## Downloading Resources

### Course Files

1. Scroll to the "Course Files" section
2. Click on any file name
3. File downloads automatically

### Lesson Resources

1. Navigate to a specific lesson
2. View attached resources in the lesson details
3. Click resource names to download

### Resource Information

Each resource shows:
- **Filename**: Name of the file
- **Size**: File size
- **Type**: File type (PDF, DOCX, etc.)
- **Created Date**: When it was uploaded
- **Created By**: Who uploaded it

## Link Previews

When course content includes links, DesQTA automatically generates previews showing:
- **Title**: Page title
- **Description**: Page description
- **Image**: Preview image
- **URL**: Link address

Click the preview to open the link in your browser.

## Searching Courses

### Search Functionality

1. Use the search bar at the top
2. Type subject codes, titles, or keywords
3. Results filter automatically

### Search Tips

- Search by subject code (e.g., "ENG")
- Search by subject name (e.g., "English")
- Search is case-insensitive
- Results update as you type

## Course Organization

### Active vs Archived

- **Active Subjects**: Currently enrolled courses
- **Archived Subjects**: Past courses (in "Other Folders")

### Subject Colors

Each subject has a unique color used throughout DesQTA:
- Timetable entries
- Assessment cards
- Course headers
- Visual identification

## Lesson Schedule Features

### Weekly View

See lessons organized by:
- **Term**: Current academic term
- **Week**: Week within term
- **Lessons**: All lessons for that week

### Lesson Details

Each lesson shows:
- **Period**: Class period number
- **Date**: Lesson date
- **Time**: Start and end times
- **Content**: Lesson description and materials
- **Resources**: Attached files

### Navigation Controls

- **Previous Week**: Navigate to earlier weeks
- **Next Week**: Navigate to later weeks
- **Term Selector**: Switch between terms
- **Overview Toggle**: Switch between schedule and overview

## Tips for Using Courses

### Daily Routine

1. **Morning**: Check today's lessons in Schedule tab
2. **Before Class**: Review lesson content and resources
3. **After Class**: Download any resources you need

### Organization

- **Bookmark Subjects**: Frequently accessed subjects appear at top
- **Download Early**: Download resources when you have internet
- **Check Schedule**: Use Schedule tab to plan ahead

### Efficiency

- **Use Search**: Quickly find subjects by code or name
- **Expand Folders**: See all subjects at once
- **Navigate Lessons**: Use week navigation to plan study time

## Troubleshooting

### Course Not Loading

**Check:**
- Internet connection
- SEQTA session is active
- Subject is available in SEQTA

**Solutions:**
- Refresh the page
- Check SEQTA directly
- Verify subject enrollment

### Resources Won't Download

**Possible causes:**
- File permissions
- Network issues
- File size too large

**Solutions:**
- Check internet connection
- Try again later
- Contact teacher if persistent

### Schedule Not Showing

**Check:**
- Term selector is set correctly
- Subject has scheduled lessons
- Date range is correct

**Solutions:**
- Select different term
- Check SEQTA for schedule
- Verify lesson dates

### Missing Content

**Possible causes:**
- Content not published yet
- Permissions issue
- Cache needs refresh

**Solutions:**
- Refresh the page
- Clear cache in Settings
- Check SEQTA directly
