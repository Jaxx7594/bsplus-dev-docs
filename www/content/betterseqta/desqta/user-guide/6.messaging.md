## Messaging Overview

DireQt is DesQTA's integrated messaging system that connects directly to SEQTA's messaging system. It allows you to send and receive direct messages with teachers, classmates, and other SEQTA users. No additional account setup is required - it uses your existing SEQTA login.

## Accessing Messages

Navigate to Messages from:
- **Sidebar Menu**: Click "Messages" in the navigation sidebar
- **Dashboard**: Click "Messages Preview" widget
- **Direct URL**: `/direqt-messages`

## No Account Setup Required

DireQt messaging works automatically with your SEQTA account. As long as you're logged into DesQTA with your SEQTA credentials, you can immediately start sending and receiving messages. No BetterSEQTA Cloud account or additional setup is needed.

## Messaging Interface

### Layout

The messaging interface consists of:

**Left Sidebar:**

- New message button

**Main Area:**
- Message thread

**Right Sidebar (Desktop):**
- Conversation details

### Mobile View

On mobile devices:

- Swipe to navigate
- Touch-optimized interface
- Bottom navigation tabs

## Starting a Conversation

### New Message

1. Click "New Message" button (plus icon)
2. Search for recipient by:
   - Username
   - Display name
   - Email
3. Select recipient
4. Start typing your message

### Composing Messages

**Text Messages:**
- Type in the message input box
- Press Enter to send
- Shift+Enter for new line
- Supports emojis and formatting

**Message Features:**
- **Formatting**: Bold, italic, underline
- **Emojis**: Full emoji support
- **Links**: Auto-detected and clickable

## Replying to Messages

### Reply to Specific Message

1. Hover over a message
2. Click "Reply" button
3. Your reply quotes the original message
4. Type your response
5. Send

### Reply Threading

Replies maintain conversation context:
- Original message quoted
- Thread visual indicator
- Easy to follow conversations

## Attachments

### Sending Files

1. Click attachment icon (paperclip)
2. Choose file from your device
3. File uploads automatically
4. Add message (optional)
5. Send

**Supported File Types:**
- Images (JPG, PNG, GIF, etc.)
- Documents (PDF, DOC, DOCX, etc.)
- Archives (ZIP, RAR, etc.)
- Other common file types

**File Size Limits:**
- Maximum file size varies
- Large files may take time to upload
- Progress indicator shows upload status

### Viewing Attachments

1. Click attachment in message
2. File opens in viewer or downloads
3. Images display inline
4. Documents open in appropriate app

## Direct Messages Only

DireQt currently supports direct messages only. Group messaging functionality is not available and has been disabled. All messaging is done through one-on-one conversations with individual SEQTA users.

## RSS Feed Support

DireQt includes support for RSS feeds, allowing you to subscribe to news feeds and updates directly within your messaging interface.

### Adding RSS Feeds

1. Click on RSS folder in sidebar
2. Add RSS feed URL
3. Feed appears in your folder list
4. Updates automatically refresh

### RSS Feed Features

- **Automatic Updates**: Feeds refresh periodically
- **Real-time Content**: Always shows latest posts
- **No Caching**: RSS content is always fresh
- **Folder Organization**: RSS feeds appear in separate folder

### Managing RSS Feeds

- **View Feed**: Click RSS folder to see posts
- **Read Articles**: Click individual posts
- **Remove Feed**: Delete RSS folder to unsubscribe

## Message Organization

**Default Folders:**
- **Inbox**: All received messages

- **Archived**: Archived conversations
- **Trash**: Deleted messages

**Custom Folders:**
- Create custom folders
- Move conversations

### Search

Find messages quickly:

1. Use search bar at top
2. Search by:
   - Sender name
   - Message content
   - Date
   - Keywords
3. Results filter in real-time

### Filtering

Filter conversations:
- **Unread**: Show only unread messages
- **Starred**: Show favorited conversations

## Privacy & Security

### Blocking Users

Block unwanted users:

1. Open conversation
2. Click user's name
3. Select "Block User"
4. User can no longer message you

### Reporting

Report inappropriate messages:

1. Open message
2. Click "Report" option
3. Select reason
4. Submit report

### Privacy Settings

Messaging permissions are controlled by your SEQTA account settings. Contact your school administrator if you need to adjust who can message you.

## Troubleshooting

### Messages Not Sending
- Check internet connection
- Verify SEQTA session is active
- Try refreshing page
- Check file size limits

### Not Receiving Messages
- Check notification settings
- Verify SEQTA login is active
- Check spam/filtered folders
- Refresh conversation list

### Attachments Not Working
- Check file size limits
- Verify file type is supported
- Try different file
- Check internet connection
