## Settings Overview

BetterSEQTA-Plus provides extensive customization options to tailor your SEQTA Learn experience.

## Accessing Settings

### Method 1: Extension Popup

1. Click the BetterSEQTA-Plus icon in your browser toolbar
2. Click "Settings" in the popup menu
3. Settings page will open

### Method 2: Right-Click Menu

1. Right-click on any SEQTA Learn page
2. Select "BetterSEQTA-Plus Settings"
3. Settings page will open

## Settings Categories

### Appearance

Customize the visual appearance of SEQTA Learn:

- **Theme** - Choose light, dark, or system theme
- **Accent Color** - Select your preferred accent color
- **Font Size** - Adjust text size
- **Layout** - Customize page layout
- **Animations** - Enable/disable animations

### Features

Toggle individual features:

- **Dashboard Enhancements** - Enable dashboard improvements
- **Assessment Tools** - Enable assessment features
- **Timetable Features** - Enable timetable enhancements
- **Messaging** - Enable messaging improvements
- **Study Tools** - Enable study features

### Behavior

Configure how features behave:

- **Auto-save** - Automatically save changes
- **Notifications** - Configure notification settings
- **Keyboard Shortcuts** - Customize shortcuts
- **Default Actions** - Set default behaviors

### Privacy

Privacy and data settings:

- **Data Collection** - Control data collection
- **Analytics** - Enable/disable analytics
- **Storage** - Manage stored data
- **Permissions** - Review permissions

## Common Settings

### Enable Dark Mode

1. Open Settings
2. Go to Appearance
3. Select "Dark" theme
4. Changes apply immediately

### Customize Colors

1. Open Settings
2. Go to Appearance
3. Click "Accent Color"
4. Choose your color
5. Preview and save

### Enable/Disable Features

1. Open Settings
2. Go to Features
3. Toggle features on/off
4. Changes apply immediately

## Advanced Settings

### Developer Options

For advanced users:

- **Debug Mode** - Enable debug logging
- **Experimental Features** - Try beta features
- **API Access** - Configure API settings
- **Custom Scripts** - Add custom scripts

### Import/Export

- **Export Settings** - Save your configuration
- **Import Settings** - Load saved configuration
- **Reset to Defaults** - Restore default settings

## Settings Sync

### Sync Across Devices

BetterSEQTA-Plus can sync settings:

1. Enable sync in settings
2. Sign in with your account
3. Settings sync automatically
4. Access from any device

## Troubleshooting Settings

### Settings Not Saving

- Check browser permissions
- Ensure storage is available
- Try refreshing the page
- Clear browser cache if needed

### Settings Reset

- Settings may reset after updates
- Export settings before updates
- Import settings after updates
- Check for migration guides
