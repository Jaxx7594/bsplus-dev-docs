## Installing BetterSEQTA-Plus

BetterSEQTA-Plus is a browser extension that enhances SEQTA Learn with additional features and improvements.

## Supported Browsers

BetterSEQTA-Plus is available for:

- **Chrome/Chromium** (Chrome, Edge, Brave, etc.)
- **Firefox**
- **Safari** (coming soon)

## Installation Methods

### Chrome/Chromium (Chrome Web Store)

1. Visit the [Chrome Web Store](https://chrome.google.com/webstore)
2. Search for "BetterSEQTA-Plus"
3. Click "Add to Chrome"
4. Confirm the installation
5. The extension will be installed and ready to use

### Firefox (Firefox Add-ons)

1. Visit the [Firefox Add-ons](https://addons.mozilla.org) website
2. Search for "BetterSEQTA-Plus"
3. Click "Add to Firefox"
4. Confirm the installation
5. The extension will be installed and ready to use

### Manual Installation (Development)

For development builds or manual installation:

#### Chrome/Chromium

1. Download the extension files from GitHub
2. Open Chrome and navigate to `chrome://extensions/`
3. Enable "Developer mode" (toggle in top right)
4. Click "Load unpacked"
5. Select the extension directory
6. The extension will be loaded

#### Firefox

1. Download the extension files from GitHub
2. Open Firefox and navigate to `about:debugging`
3. Click "This Firefox"
4. Click "Load Temporary Add-on"
5. Select the `manifest.json` file from the extension directory
6. The extension will be loaded

## First Launch

After installation:

1. Navigate to your SEQTA Learn instance
2. The extension will automatically activate
3. You may see a welcome message or notification
4. Check the extension icon in your browser toolbar

## Verifying Installation

To verify BetterSEQTA-Plus is installed:

1. Look for the BetterSEQTA-Plus icon in your browser toolbar
2. Click the icon to open the extension popup
3. You should see the extension menu and options
4. Navigate to SEQTA Learn - enhanced features should be visible

## System Requirements

- **Browser:** Latest version of Chrome, Firefox, or Safari
- **SEQTA Learn:** Any modern SEQTA Learn instance
- **Permissions:** The extension may request permissions for:
  - Access to SEQTA Learn pages
  - Storage for settings and preferences

## Troubleshooting

### Extension Not Appearing

- Refresh your browser
- Check if the extension is enabled in browser settings
- Verify you're on a SEQTA Learn page

### Features Not Working

- Ensure you're logged into SEQTA Learn
- Check extension permissions in browser settings
- Try disabling and re-enabling the extension

### Installation Errors

- Ensure your browser is up to date
- Check that you have permission to install extensions
- Try installing from the official store instead of manual installation
