## Welcome to BetterSEQTA-Plus

After installing BetterSEQTA-Plus, follow this guide to get started with your first session.

## Initial Setup

### 1. Navigate to SEQTA Learn

1. Open your browser
2. Navigate to your school's SEQTA Learn instance
3. Log in with your SEQTA credentials
4. BetterSEQTA-Plus will automatically activate

### 2. Extension Activation

When BetterSEQTA-Plus detects SEQTA Learn:

- The extension icon will show as active
- Enhanced features will be available
- You may see a welcome notification

### 3. First-Time Setup Wizard

On first launch, you may see a setup wizard:

1. **Welcome Screen** - Introduction to BetterSEQTA-Plus
2. **Feature Tour** - Overview of available features
3. **Settings** - Configure initial preferences
4. **Done** - Start using BetterSEQTA-Plus

## Extension Popup

Click the BetterSEQTA-Plus icon in your browser toolbar to access:

- **Quick Settings** - Toggle features on/off
- **Feature List** - View available enhancements
- **Settings** - Access full settings page
- **About** - Version and information

## Key Features Available

BetterSEQTA-Plus enhances SEQTA Learn with:

- **Enhanced UI** - Improved interface and styling
- **Quick Actions** - Faster access to common tasks
- **Customization** - Personalize your SEQTA experience
- **Performance** - Optimized loading and interactions
- **Additional Tools** - Extra features not in standard SEQTA

## Verifying Features

To verify features are working:

1. Look for visual changes to the SEQTA interface
2. Check for new buttons or menu items
3. Try enhanced features like quick actions
4. Verify settings are being saved

## Troubleshooting First Launch

### Extension Not Activating

- Ensure you're on a SEQTA Learn page
- Check that the extension is enabled
- Refresh the page
- Try logging out and back in

### Features Not Visible

- Some features may require specific SEQTA pages
- Check extension settings for feature toggles
- Verify you have the latest version installed

### Setup Wizard Not Appearing

- The wizard may have been dismissed
- Access settings from the extension popup
- Reset extension data if needed
